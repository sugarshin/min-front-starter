# min-front-starter

private minimal frontend starter

## Getting started

```shell
mkdir project-name && cd $_
git clone --depth 1 git@github.com:sugarshin/min-front-starter.git .
sh init.sh
```

scripts

```shell
npm start
```

use

* beefy
* browserify
* coffeeify
* cssify
* virtual-dom
